ScribbleSense AI
AI that converts handwritten notes into structured study material.

Features
Upload handwritten notes image
AI extracts structured notes
Generates 3-line summary
Creates quiz automatically
Extracts key terms
Personalized study tip
Download as PDF

How it works
Upload handwritten notes photo
AI analyzes content
Generates notes, summary, quiz and key terms
Download as PDF for revision

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python Flask
AI: Google Gemini API
PDF: html2pdf.js

Demo Flow
Upload → Analyse → Get notes → Download PDF

Project Goal
Students often write messy handwritten notes.
This tool converts them into structured, revision-ready study material instantly.
